Windows 10 Installation & System Recovery

By: Holly
Date: June 2025
Location: Guanacaste, Costa Rica

Overview

I performed a full system recovery and Windows 10 clean installation on an older machine running Windows Server 2022 with only command-line access. This required USB troubleshooting, BIOS configuration, and low-level media handling — all tackled step by step and independently.

Key Skills

  • Bootable media creation (Rufus, ISO prep)
  • Disk partitioning and Secure Boot configuration
  • Command-line system handling and recovery setup
  • Cloud backup workflows (Google Drive + manual)

Challenges Solved

  • USB write-protection errors and media tool failures
  • UEFI & BIOS tweaks including Secure Boot deactivation
  • Revoked bootloader issues bypassed with ISO validation
  • Windows ISO download access on unsupported OS using user-agent spoofing

Outcome

The machine was successfully restored to Windows 10, data was preserved, and I developed a deeper knowledge of PC infrastructure and deployment strategy — a huge win in personal tech growth!
